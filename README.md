dotfiles
========
## Pre-installation

- Create and add an [SSH key](https://github.com/settings/keys) to GitHub and clone this repo.
- Run the script to show all the commands. Source the script to execute.

```
git clone git@github.com:initrc/dotfiles.git
./dot.sh
```

## Linux
```
dot-linux-dep
dot-vim
dot-vim-config
dot-shell
dot-git-config
dot-git-scm-breeze
```
