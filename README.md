dotfiles
========
# Pre-installation

## Github
- Create and add an SSH key to GitHub(https://github.com/settings/keys) and clone this repo.
- Run the script to show all the commands. Source the script to execute.

```
git clone git@github.com:initrc/dotfiles.git
./dot.sh
```

# Linux
```
dot-linux-dep
```
